Python Go TypeScript AWS LLMs Distributed Systems ML Infrastructure

I'm a Software Development Engineer at Amazon Ads, where I work on ML infrastructure, LLM creative compliance evaluation systems, and distributed notification services. I enjoy building systems that are both highly scalable and deeply principled.

My research interests live at the intersection of serverless computing and applied AI. I've published work on serverless function search, open-source ecosystems, and the automation of serverless infrastructure. These questions drive me: how do we make cloud systems smarter, more observable, and easier to build?

Outside of work, I run IntroToAlgo — a blog I've been writing since 2017 about algorithms, data structures, and computing fundamentals.
